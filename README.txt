Actual non-experimental answers for my Intel i7 4770k:
    L1 Data cache = 32 KB, 64 B/line, 8-WAY.
    L1 Instruction cache = 32 KB, 64 B/line, 8-WAY.
    L2 cache = 256 KB, 64 B/line, 8-WAY
